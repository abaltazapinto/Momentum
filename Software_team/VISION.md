# Momentum

Momentum is a modern personal productivity application built incrementally.

The goal is not only to build a task manager, but to demonstrate professional software engineering practices.

Core principles:

- Incremental development
- Small pull requests
- Clean architecture
- Accessibility
- Testing
- Documentation
- Real-world Git workflow
- AI-assisted development

## Roadmap

V1  Core Task Manager
V2  Categories & Priorities
V3  Due Dates & Search
V4  Authentication
V5  Drag & Drop
V6  Dark Mode
V7  Offline + PWA
V8  Notifications
V9  Analytics Dashboard
V10 AI Productivity Assistant
